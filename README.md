# JANSORi

**랜덤 알람 기능을 제공하는 안드로이드 어플리케이션**

[개인 프로젝트]
해야 할 일이 뭔지 기억하지 못하는 사람들,<br>
멍하니 나도 모르게 해버리는 나쁜 습관들을 가진 사람들을 위한 어플리케이션을 제작한다<br><br>

「JANSORi」는 시간에 맞추어 알람이 울리는 기본 알람 기능과 더불어 <br>
습관 개선을 위한 랜덤 알람 기능을 제공하여 개인 비서처럼 사용자를 서포트한다<br><br>
